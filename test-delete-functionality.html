Manual Test Instructions

  1. Open index.html in a web browser
  2. Add several todos using the "Add Todo" form
  3. Click the "Delete" button on any todo
  4. Verify that a confirmation dialog appears asking "Are you sure you want to delete this todo?"
  5. Click "OK" to confirm deletion
  6. Verify that the todo is removed from the list
  7. Repeat for multiple todos to ensure consistent functionality
  8. Verify that when all todos are deleted, the "No todos yet" message appears
